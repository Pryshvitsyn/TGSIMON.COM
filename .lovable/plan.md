

## Исправление сборки — убрать автодеплой

### Проблема
Изменение `outDir` на `cloudflair-site` сломало сборку в Lovable (система ожидает папку `dist`).

### Что будет сделано

1. **`vite.config.ts`** — убрать блок `build.outDir`, вернуть стандартную папку `dist`
2. **Удалить `.github/workflows/deploy-cloudflare.yml`** — автоматический деплой не нужен

Файл `public/_redirects` оставим — он пригодится для корректной работы React Router на Cloudflare Pages когда вы будете деплоить вручную.

### Результат
Сборка снова заработает. Когда вам нужно задеплоить на Cloudflare, вы просто скачиваете/клонируете репозиторий с GitHub, делаете `npm run build` локально, и загружаете папку `dist` в Cloudflare Pages Dashboard вручную.
